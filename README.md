# AI\_Research

This repository contains the data and documentation for a research study evaluating the effectiveness of Generative AI tools.



The project is organized into the following directories:

&#x20;\*\*AI\_Research/\*\*: Main directory for all research materials.

&#x20;   \* `/literature\_review`: Academic sources and articles.

&#x20;   \* `/quantitative\_analysis`: Survey data and analysis scripts.

&#x20;   \* `/qualitative\_analysis`: Interview transcripts and protocols.

&#x20;   \* `/drafts\_and\_reports`: Research drafts and final reports.



\## How to Contribute

2\. Create a feature branch (`git checkout -b feature-name`).
